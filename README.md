Rick got tha drapes on lock.

This is a hubot script that returns random images of Rick Owens, The Dark Prince of the Geometric Deconstruction.

When the user types "<botname> drape" the script returns a single random image of Rick.

When the user types "<botname> rick bomb" the script returns multiple random images of Rick.
